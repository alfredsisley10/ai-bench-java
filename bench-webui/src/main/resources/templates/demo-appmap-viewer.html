Official AppMap viewer not vendored yet. The same Vue.js viewer that VS Code, the Confluence app, and app.land use is shipped in @appland/components, but it depends on Vue, Vuex, marked, mermaid, sql-formatter, highlight.js, dom-to-svg, dompurify, and ~10 other packages — too much for a CDN drop-in. Run ./scripts/vendor-appmap-viewer.sh from the repo root to bundle everything into a single browser-ready IIFE under static/appmap/components.iife.js; the viewer auto-detects it and replaces the fallback below with the full interactive experience (filter, sequence diagram, drill-down, pan/zoom). Until then, download the raw JSON and open it in the AppMap VS Code extension — that's the same viewer.
