How to fix

Each option below has a button that runs the fix in-process. Enter the path to your banking-app checkout (or the repo root that contains it) and pick an option:

  1. Option A — set the env var (recommended). Stop bench-webui, export AI_BENCH_BANKING_APP to the absolute path of your banking-app checkout, then restart: 
    # PowerShell
Stop-Process -Id (Get-Content $HOME\ai-bench\bench-webui.pid) -Force
$env:AI_BENCH_BANKING_APP = "C:\path\to\ai-bench-java\banking-app"
.\scripts\start-bench-tools.ps1
    Persists to ~/.ai-bench/banking-app-path.txt and applies in-process — no restart needed.
