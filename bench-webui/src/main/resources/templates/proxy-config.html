Proxy & TLS settings

Outbound connections from the WebUI (GitHub, JIRA, AppMap agent, Gradle subprocesses, banking-app health probe) all honor these settings. Proxy host/port is also written to ~/.gradle/gradle.properties so Gradle builds launched outside the WebUI inherit the same proxy.

Configure

Insecure-SSL scope: applies to all WebUI-originated HTTP clients (GitHub, JIRA, banking-app probes, AppMap agent calls). For Gradle subprocesses the WebUI also passes -Dcom.sun.net.ssl.checkRevocation=false on the command line, which most JVM-based resolvers honor — but Gradle plugins that bring their own HTTP client may still require a CA cert installed in the JVM truststore.
