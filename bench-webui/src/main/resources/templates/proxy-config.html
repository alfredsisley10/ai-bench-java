Proxy & TLS settings

Outbound connections from the WebUI (GitHub, JIRA, AppMap agent, Gradle subprocesses, banking-app health probe) all honor these settings. Proxy host/port is also written to ~/.gradle/gradle.properties so Gradle builds launched outside the WebUI inherit the same proxy.

Detect from OS & VSCode

Reads proxy settings from the OS (macOS networksetup / Windows registry HKCU Internet Settings) and from VSCode user settings.json (http.proxy, http.proxyAuthorization, http.noProxy). Pre-fills the form below — you still have to click Save & apply to commit.
