Verify connectivity

Probes the endpoints Gradle needs to bootstrap a build — services.gradle.org (wrapper distribution), repo.maven.apache.org (Maven Central), api.foojay.io (JDK toolchain auto-provisioning), your configured Artifactory mirror, and the corporate proxy itself. Each probe runs through whatever proxy + auth + mirror settings you've saved above, so a green sweep here means the live config will work the next time you click Run gradlew --version on the demo page.

~5–15 seconds for the full sweep.

Detect from OS & VSCode

Reads proxy settings from the OS (macOS networksetup / Windows registry HKCU Internet Settings) and from VSCode user settings.json (http.proxy, http.proxyAuthorization, http.noProxy). Pre-fills the form below — you still have to click Save & apply to commit.
