API docs

Interactive OpenAPI 3 spec for every REST endpoint the WebUI exposes — generated from the live @GetMapping / @PostMapping annotations by springdoc, so it's always in sync with the running build (no hand-maintained table to drift). Two groups available in the Swagger dropdown:

  • demo — the public-facing demo issue / run / grade endpoints used by the dashboard and the demo page.
  • internal — admin/management endpoints (proxy config, JIRA / GitHub auth, banking-app lifecycle, etc.).

Use the Try it out button on any endpoint to call it directly with the current session's auth.
